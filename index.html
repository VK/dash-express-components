Dash Express Components

Widgets to bring Plotly Express style plots to Dash


Example uses dash-lumino-components for MDI layout.

Try it

Install dependencies

$ pip install dash-express-components

and start with quickly editable graphs:

import dash_express_components as dxc
app.layout = html.Div([

    # add a plot dxc.Configurator
    html.Div([
        dxc.Configurator(
            id="plotConfig",
            meta=meta,
        ),
    ], style={"width": "500px", "float": "left"}),

    # add an editable dxc.Graph 
    html.Div([
        dxc.Graph(id="fig",
                  meta=meta,
                  style={"height": "100%", "width": "100%"}
                 )],
        style={"width": "calc(100% - 500px)", "height": "calc(100vh - 30px)",
               "display": "inline-block", "float": "left"}
    )
])

Develop

  1. Install npm packages

    $ npm install

  2. Create a virtual env and activate.

    $ virtualenv venv
$ . venv/bin/activate

    Note: venv\Scripts\activate for windows

  3. Install python packages required to build components.

    $ pip install -r requirements.txt

  4. Build your code

    $ npm run build

  5. Run the example

    $ python usage.py
